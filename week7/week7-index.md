# Week 7

<SessionResources :week="7" />

This week focuses on **efficient data handling, system-level understanding, and web security fundamentals**.

You will explore how **search algorithms optimize data retrieval**, how **memory architecture impacts performance**, and how databases integrate with web applications.<br>
It also introduces **real-world security vulnerabilities** and how to prevent them.

By the end of this week, you should be able to:

* choose appropriate **search algorithms** for different scenarios
* understand how **memory hierarchy affects system performance**
* integrate **SQL with web applications**
* identify and prevent **common web security issues**

### Topics Covered

* **[Data Search Algorithms](7-data-search-algo.md)**
  Linear vs Binary Search, time complexity, and practical use cases

* **[Memory Architecture](7-memory-architecture.md)**<br>
  Memory hierarchy, cache levels, and performance trade-offs

* **[SQL & Web Integration](7-SQL-web.md)**
  Using SQL in web apps, CRUD operations, and backend connectivity

* **[Web Security](7-web-security.md)**
  SQL injection, input validation, and secure coding practices
